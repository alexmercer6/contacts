# Contacts

## Description

Simple contacts app made in React utilising the jsonplaceholder users API

## Getting Started

### Dependencies

* Animate.css
```
import "animate.css"
```
* Axios
```
import axios from "axios"
```
* React-router-dom
* React-icons
* React-spinners

### Installing

* npm install to load dependencies from package.json
```
npm install
```
* npm start to run the project locally
```
npm start
```
